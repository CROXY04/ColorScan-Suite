To start the application:
1. Run the ColorScan.exe file
2. You'll have two languages to chose from
3. When selected you can chose the different tools included in your version of ColorScan as well as be able to read the impressum

ColorCam - The tool ColorCam is capable of opening the webcam and when double clicked on a pixel, it shows the RGB values with it respective names.

ColorPic - The tool ColorPic is capable of opening png and jpg files. Which then can be used to get RGB values and its name when double clicked.
